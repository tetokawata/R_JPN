Chapter 5 予測関数の推定

5.1 データの導入

data("NMES1988",
     package = "AER")

raw <- NMES1988

5.2 データ分割

  • ここでは5個のデータに分割する。
group <- sample(1:5,
                size = nrow(raw),
                replace = TRUE)

5.3 OLS

  • 線形予測関数\(f(X)=\beta_0 + \beta_1X_1+...+\beta_LX_L\)を仮定し、最小二乗法にて推定する。
fit <- lm(visits ~ .,
          data = raw[group != 1,])

coef(fit)

5.4 LASSO

  • glmentパッケージ(Friedman et al. 2021)を利用
library(glmnet)

Y <- raw$visits

X <- model.matrix(visits ~ -1 + .,
                  data = raw)

cv <- cv.glmnet(x = X[group != 1,],
                y = Y[group != 1],
                alpha = 1)

fit <- glmnet(x = X[group != 1,],
              y = Y[group != 1],
              alpha = 1,
              lambda = cv$lambda.min)
  • 予測モデルの確認

5.6 Bagging

  • rangerパッケージ(Wright, Wager, and Probst 2020)を利用
library(ranger)

fit <- ranger(x = X[group != 1,],
              y = Y[group != 1],
              num.trees = 2000,
              mtry = ncol(X))
  • 予測値の計算
Y.hat <- predict(fit,X)$predictions
  • テストデータへの適合
mean((Y - Y.hat)[group == 1]^2)
## [1] 35.38
  • 訓練データへの適合
mean((Y - Y.hat)[group != 1]^2)
## [1] 9.016629
